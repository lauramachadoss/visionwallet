# VisionWallet

VisionWallet é um projeto pessoal desenvolvido com o objetivo de praticar e aprimorar conhecimentos em desenvolvimento web utilizando HTML, CSS e JavaScript.

A aplicação funciona como um planejador financeiro simples, permitindo o registro de receitas e despesas, o acompanhamento do saldo atual e a visualização de um histórico de transações de forma organizada.

## Sobre o Projeto

Este projeto foi criado para fins de estudo e aprendizado, com foco em conceitos fundamentais do desenvolvimento front-end, como manipulação do DOM, armazenamento local de dados e construção de interfaces responsivas.

## Funcionalidades

* Cadastro de receitas e despesas
* Categorização de transações
* Cálculo automático do saldo
* Histórico financeiro
* Exclusão de registros
* Armazenamento local com LocalStorage
* Interface responsiva

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
